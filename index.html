Android ADB Tool

Software developed in Java using JavaFX whilst also using windows CMD.

Having owned several Android devices there are not many existing tools that take advantage of Android ADB commands over Wi-Fi in a GUI based solution, hence this solution was produced. It offers user the ability to send or pull files from multiple Android devices whilst also offering the option to install APK files in an easy and friendly to use solution.

  • Developed With: Java using JavaFX for the GUI whilst also using Android ADB executables for execution of commands.
  • Category: Software Development

Music & Video Download

Download your favourite audio and music with ease.

This solution allows the users to download music and audio from YT whilst also supporting the download of playlists or mass URLS with an easy-to-use solution. This is built by using Python with an external library Pytube. I have taken this library and implemented functions such as search and download video / audio from YT whilst also implementing. a mass download feature which consists of reading local text files which will contain the URL.

  • Solution information: Built with Python using the Pytube external library
  • Category: Software Development

File Backup Solution

Simple one click file backup solution.

Simple one click file backup solution, the user specifies folders to backup and desired location in the configuration file and then proceed to use the tool to perform the backups. This was developed in Java which consisted of strong knowledge of logical operations, looping and constant data checks to ensure the correct files were being backed up.

  • Developed with: Java using a JavaFX GUI
  • Category: Software Development

Text Patching Tool

Simple tool to replace strings within files based on the config used.

Unique but useful tool which can replace values within a file based on the values implemented in the configuration file, this config file specifies which files need editing and what values need to be replaced and what with. This tool was developed in Java using JavaFX whilst consisting of strong understanding of logical approaches and controlled looping.

  • Developed with: Java using IntelliJ with JavaFX
  • Category: Software Development
